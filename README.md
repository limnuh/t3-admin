Migrate:
`npx prisma migrate dev`

Install:
`npm i`
`npx prisma db push`

Run in dev mode (with sst):
`npm run dev`

Run in dev mode (with next):
`npm run dev-next`

Build (with sst):
`npm run build`

Run cron:
`npm run cron`

Run in docker (new)
`docker compose up -d`
Stop:
`docker compose down`
Run in older docker:
`docker-compose up -d`
Stop:
`docker-compose down`
